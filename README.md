# Library Management System (C Language)

A console-based Library Management System developed in C using a modular multi-file structure. This project helps manage library operations such as adding, searching, issuing, returning, modifying, and deleting book records. It demonstrates the use of file handling, structures, functions, and modular programming concepts in C.

## Features
- Add new books
- Display all books
- Search books by ID or title
- Issue books
- Return books
- Modify book details
- Delete book records
- Store data using File I/O
- Menu-driven user interface

## Technologies Used
- C Programming
- Structures
- Functions
- File Handling
- Modular Programming
- GCC Compiler
